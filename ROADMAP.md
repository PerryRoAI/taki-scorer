# Roadmap

Planned features and improvements for Taki Scorer.

## Near-term

- [ ] **Score tracking across games** — cumulative points per player based on finishing position
- [ ] **Player stats** — win count, average finishing rank, games played
- [ ] **Custom card starting count** — let players choose a starting hand size other than the default
- [ ] **Confetti animation** — visual celebration when the first player finishes

## Medium-term

- [ ] **PWA / installable app** — add a web manifest and service worker so the app installs like a native app on iOS and Android
- [ ] **Multi-language support** — English UI alongside the current Hebrew
- [ ] **Share results** — export or share tournament results as an image or text
- [ ] **Timer per turn** — optional countdown timer to keep turns moving

## Long-term / Ideas

- [ ] **Game history** — persist tournament logs across browser sessions with a full history view
- [ ] **Custom themes** — light mode, alternate color schemes
- [ ] **Multiplayer sync** — real-time score sharing across devices on the same local network

## Completed

- [x] 2–10 player support with customizable names
- [x] Card counter with tap-to-decrement
- [x] Auto-finish last player
- [x] Undo last drop
- [x] Move log with timestamps
- [x] Tournament log (winners per game)
- [x] Sound effects with per-type toggles
- [x] Haptic feedback toggle
- [x] localStorage persistence
- [x] Mobile-first RTL Hebrew UI
